# aws-ivs-broadcast-from-browser
Broadcast from browser for AWS IVS

Step 1. Install ffmpeg

>choco install ffmpeg

>apt-get install ffmpeg


Step 2. Start Server

(Enter streamKey and rtmpUrl)

>node server.js


Step 3. Go Live

>www/index.html
